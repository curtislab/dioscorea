Research Publications

We will eventially provide a news feed asociated with Dioscorea related new publications.

The most immediate source of Dioscorea references can be accessed by creating a FREE account on no-cost web-based reference manager MENDELEY and then adding the "Dioscorea.org" Mendely Group. Note that since this is a PUBLIC group, it is not possible to attach the PDF files; however, this allows anyone to gain access to the databased as well as the associated comments and curated citation metafile data. Note that you can directly access the online MENDELEY database by placing the following URL into your browser www.mendeley.com/groups/8056931/dioscorea-org/

If it is found useful, a periodic listing of references based on specific keywords wil be generated and posted for quick reference. Suggested keywords are as follows:
  • somatic embryogensis
  • dormancy
  • genomics AND bioinformatics

We suggest that it would be very useful to be able to distribute URL references to self-achirved submitted papers as well. The tools associated with understanding self-archive are explained at the web site archive.it which is a interface to simplify the SHERPA/RoMEO database of publishers' copyright and self-archiving policies.
